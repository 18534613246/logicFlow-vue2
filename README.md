# vue2-logicflow
npm install
npm run serve
## 项目简介
vue2+logicFlow1.2开发的可拖拽的图形编辑器（lf2.0版本后续迭代哈）

### 代码已使用的功能Demo
1，基于vue的自定义节点，可与业务代码进行通信和交互  
2，自定义桩和线条，贝塞尔曲线  
3，比较美观的前端页面交互
